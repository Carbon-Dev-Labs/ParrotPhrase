Parrot Phrase

Change up the way that you write with Artificial Intelligence.

Get Started

Github

Visit the Paraphraser for more options

Parrot Phrase is a text editing tool that uses machine learning and artificial intelligence to help make your writing sound more formal or more human.
When paraphrasing, there are different tones and formats you can adjust to make your writing have a tone to convey meaning.
If you are using Chat GPT to write a letter or something to send to someone, Parrot Phrase can use synonyms to make the writing stronger and more formal.
